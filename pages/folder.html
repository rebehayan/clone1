Folder Guide

파일 및 폴더 네이밍 규칙 예
분류 예제 설명
Folder root 작업 현황 / 가이드 / index 파일
pages index파일 포함 업무별 폴더 생성가능.
js 목적별 폴더 생성 가능
css 디렉토리별 파일 생성 가능
scss 목적에 따라 디렉토리 생성
images 네이밍별 폴더 생성 가능
HTML pageid.html 화면 코드 사용
pageid_p.html 팝업 파일 사용 ( 파일명 + p )
pageid_t.html 탭메뉴 파일 사용 ( 파일명 + t )
CSS pagename.css 1뎁스명을 사용
reset.css 브라우저 호환성을 위하여 기본 css 속성을 정의한다.
style.css 전역에 쓰이는 컴퍼넌트와 레이아웃을 정의한다.
SCSS abstracts CSS에 영향에 없는 추상적 파일 정의
base 기본정의
components 컴퍼넌트 정의
layout 레이아웃 정의
page 페이지 정의
JS global.js 전역에 사용하는 자바스크립트를 작성한다.
setting/settingname.js 설정에 따라 작성한다.
ul/uiname.js ui 기능에 따라 작성한다.
Image ico_name.file 아이콘 이미지 정의
bu_name.file 블릿 이미지 정의
bg_name.file 배경목적으로 사용되는 이미지 정의
img_name.file 전경목적으로 사용되는 이미지 정의
@name.file 임시이미지 정의
