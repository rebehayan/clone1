Color Guide

  • 색상은 반드시 변수로 지정하여 사용한다.
  • 디자인 시스템 정의가 되어 있지 않을 경우 다음과 같은 색상명으로 정의한다.

CSS

  --white : #fff;
  --black : #000;
  --black02 : rgba(0,0,0,0.2);
  --blue : #00f;
  --gray22 : #222;
  --grayeb : #ebebeb;
  --grayf1 : #f1f1f1;

Sass(SCSS)

  $white : #fff;
  $black : #000;
  $black02 : rgba($black, 0.2);
  $blue : #00f;
  $gray22 : #222;
  $grayeb : #ebebeb;
  $grayf1 : #f1f1f1;
