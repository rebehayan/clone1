CSS Guide

기본 규칙

대소문자

모든 속성은 영문 소문자로만 작성

  /* 잘못된 예 */
.class-name {color:#FFF;}

/* 올바른 예 */
.class-name {color:#fff;}

세미콜론(;)

마지막 선언된 속성에도 세미콜론(;)을 사용한다.

  /* 잘못된 예 */
.class-name {margin-left:0; padding-left:10px}

/* 올바른 예 */
.class-name {margin-left:0; padding-left:10px;}

줄내림

  • CSS 속성마다 줄내림 한다.
  • 쉼표(,) 선택자는 줄내림 한다.
  • 단일 속성은 줄내림하지 않는다.
/* 잘못된 예 */
h5.class-name { font-size:1.25rem; letter-spacing:-1px; margin:0.5rem 0 0 0; }

/* 올바른 예 */
.title,
.box,
.block {
  padding:0.5rem;
}

h5.class-name {
  font-size:1.25rem;
  letter-spacing:-1px;
  margin:0.5rem 0 0 0;
}

.text-desc { color:#000; }

주석

  • CSS 주석 처리는 시작 주석을 작성하며 종료 주석은 작성하지 않는다.
  • 종료코드 이후에 오는 요소에 한 칸 띄어 쓰기를 허용한다.
/* 제목 */
.title-type1 {margin:3rem;}

/* 상세설명 */
.desc1 {
  padding:0.2rem 0 0 1.5rem;
  font-size:1rem;
}

빈줄

  • 객체를 구분하기 위하여 코드 그룹 간 1줄의 빈 줄을 허용한다.
  • 빈 줄의 1줄을 초과하지 않는다.
/* 잘못된 예 */
.title-type1 {margin:3rem;}
.desc1 {
  padding:0.2rem 0 0 1.5rem;
  font-size:1rem;
}

/* 올바른 예 */
.title-type1 {margin:3rem;}

.desc1 {
  padding:0.2rem 0 0 1.5rem;
  font-size:1rem;
}

선택자

공통 선택자

'*' 공통 선택자는 웹 페이지의 성능을 저하를 가져오기 때문에 사용하지 않는다.

  /* 잘못된 예 */
* {margin:0; padding:0;}

id와 class

  • id와 class를 조합하여 사용하지 않는다.
  • id를 css로 표현하기 위하여 선택자로 사용하지 않는다.
  • id는 label 또는 본문바로가기 등 내부링크에만 사용한다.
/* 잘못된 예 */
#header .lnb {margin:0; padding:0;}
#lnb {position:relative;}

/* 올바른 예 */
.lnb {position:relative;}
.lnb a {display:block;}

속성의 선언 순서

.selector{
  overflow : visible, hidden, auto;
  display : block, inline-block, flex, grid;
  position : absolute, fixed, sticky / left, right, top, bottom, z-index;
  width : min-width, max-width, clamp;
  height : min-height, max-height, clamp;
  margin
  padding
  border
  background
  font : color, line-height, font-weight, letter-spacing, font-size;
  etc : animation, transition, transform;
}

속성의 축약/약식 속성 사용

CSS 최적화를 위해 속성값을 축약 한다.

축약 표현 예시
축약 전 축약 후 설명
#ffffff #fff 6번 연속된 숫자나 문자는 3글자로 축약한다.
top:0px top:0 속성의 값이 0일 경우 단위를 표시하지 않는다.
margin:15px 15px 15px 15px margin:15px 속성의 값이 동일한 경우 축약한다.

rem 사용

  • 다양한 디바이스에 대응하기 위해 rem단위를 사용한다.
  • root font-size는 62.5%(10px)로 기준으로 한다. 
html {font-size:62.5%;}
body {font-size:1rem;}
