Align Guide

  • 부모, 자식으로 구성된 레이아웃이어야 한다.
  • 요소의 종류와 상관없이 클래스명으로만 제어한다.

요소 수평정렬

가운데정렬
<div class="align center">
  // 자식요소 삽입
</div>
좌측정렬
<div class="align left">
  // 자식요소 삽입
</div>
우측정렬
<div class="align right">
  // 자식요소 삽입
</div>

요소 수직정렬

상단정렬
<div class="align center vt">
  // 자식요소 삽입
</div>
중단정렬
<div class="align center vm">
  // 자식요소 삽입
</div>
하단정렬
<div class="align center vb">
  // 자식요소 삽입
</div>

텍스트 정렬

inline, inline-block, inline-flex, inline-grid, text 등을 정렬 한다.

텍스트 좌측정렬 
<div class="txt-left">
// 텍스트
</div>
텍스트 가운데정렬 
<div class="txt-center">
// 텍스트
</div>
텍스트 우측정렬 
<div class="txt-right">
// 텍스트
</div>
