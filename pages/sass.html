Variables

Color

  • 모든색상은 $컬러명으로 정의한다
  • 유채색은 $컬러명+숫자로 정의한다
  • 무채색은 $컬러명+컬러코드 앞두자리로 정의한다
  • 투명색은 $컬러명+투명도 숫자로 정의한다
무채색
$white: #fff;
$black: #000;
$black01: rgba($black, 0.1);
$black03: rgba($black, 0.3);
$gray29: #292d32;
$graycc: #ccc;
$grayee: #eee;

유채색
$blue: #1765fd;
$blue2: #1789ff;
$green: #4fcb8d;
$purple: #5c31d6;
$red: #fe5957;

Mixin

접근성 전용 숨김처리

@include a11y-hidden

배경이미지

@include bg(이미지명, 좌표, 배경색, 크기);
@include bg('img.svg', 10px 20px, $red, cover);

구분선 컴퍼넌트

@include bar(선 or 배경선택, 가로크기, 세로크기, 좌우여백, 배경값);

@include bar(line, 1px, 10px, 5px, $red);
@include bar(img, 10px, 10px, 5px, 'bu.svg');

말줄임표

@include ellipsis(말줄임표 줄수);
@include ellipsis(2);

Position

@include position(x축, x축값, y축, y축값, zindex값);
@include position(left, 50%, top, 50%, 10);

반응형

@include mobile{ 모바일전용 CSS삽입};

@include mobile{ 
  div { color:$red;}
}
@include tablet{
  div { color:$blue;}
}

스크롤 디자인

      @include scroll(스크롤방향, 가로크기, 색상);
@include scroll(right, 10px, $gray29);
@include scroll(bottom, 10px, $gray29);

다크테마

      @include theme{ CSS코드 삽입 };
@include theme{
  --white:red;
};
